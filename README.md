# DungeonsAndDegens

**DungeonsAndDegens** is a web app for creating RPG characters, assigning skills, items, and stats, and exploring a dynamic map. It also features tools for Dungeon Masters (DMs) to manage campaigns and quests.

## Features
- **Character Creation**: Build characters with customizable stats, skills, and items.
- **Interactive Map**: Explore a map and track adventures.
- **DM Tools**: Manage NPCs, quests, and events.
- **User-Friendly UI**: Simple interface for both players and DMs.
